Note Taking Web App

Field

Web Development

Tools

JSX

CSS

React.js

Adobe XD

Date

Oct 2020

Objective

To develop my skills in React.js, I decided to learn and make a project using this JavaScript library.
For learning, I took a LinkedIn Learning course called Become a React Developer.

Brainstorming & Initial Ideas

First, I brainstormed the possible types of web apps I could make with the limited knowledge of React.js that I had just acquired.
After deciding on a note-taking app, I began writing about the functionality that the app would have and the different components that I would need to create.
I also started making some rough wireframes of the app to get a better visual idea of the app.

Moodboard

For the UI design of the app, I collected two different kinds of images. One was modern and another a little retro.
I used this moodboard to design the interface of the app.

UI Design

Based on the moodboard, I designed these two hi-fi mockups.
I chose the second one as I felt that there are already enough modern-looking note-taking apps out there. A retro-looking note app stood out to me a little so I chose that.

Responsive Design

I used Flexbox to lay out the note cards and media queries to restrict the number of cards that can be displayed according to the device's screen width.

Live Preview

This is the final web app. Making this project taught me a lot about React.js and programming in general. Making this project made me appreciate JavaScript even more as React.js is made on JS. This could have been made in JS too but the way the code and the files were structured made a lot more sense in React.js.
