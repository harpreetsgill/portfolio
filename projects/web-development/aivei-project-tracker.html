Aievei - A project tracking website

Field

Web Development

Tools

HTML

CSS (SASS)

PHP

MySQL

JavaScript

Date

Nov, 2020

Objective

This was a self-led project that I did to understand the CRUD operations in PHP and MySQL. A lot of the code is based on a term project that I did in Term 3.

Listing the possible features

I wrote down the features that I wanted to incorporate into the website. Initially I had planned to make a feature that would allow users to add group projects and share it with other users. But as the deadline got closer, I narrowed the focus to only the essential features.

Structuring the files

To make the development process easier and efficient, I separated the features into different files and directories. All the back-end features were put inside the app directory. Different components of the front-end were split into individual files based on their functionality and put inside the partials directory.

Styling

I wanted to learn SASS so I used this project to get familiar with it. I used variables, functions, extend/inheritance and nesting features. I made use of most of these in the beginning but as the due date got closer, I started using only the things that I was comfortable with.

Challenges

I gave myself only a week to make this project. Getting stuck on something small and spending hours on it took a lot time but also hammered in my brain things I should do differently.

When trying to move the project from localhost to the live server, I couldn't get the user login session to start. After spending some hours on it and not being able to find a fix, I asked my instructor, Airrick Dunfield for help. He helped me to get the website fully running on the live server.

Live Preview

The process of making something that can be used in the real world was quite satisfying. Even though the features are very basic, this taught me a lot that I will use in the future projects.
