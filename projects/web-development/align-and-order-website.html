Objective

For the Mobile Web course in Term 2, we were required to make a front-end mobile-responsive website for an online store.
I created a fictional brand called Align & Order that sells clocks, lamps and chairs.

Wireframes

I sketched the wireframes for mobile on the paper first and then designed them in Adobe Xd.
For the tablet screen, I copied the mobile design and stretched the width of some images to fill up the whole screen. The navigation area had more space so I expanded the navigation links. Since there was more space to fit more products, I added more products in each row.
For the desktop, I stretched the hero image and the footer background to fill the whole width of the screen.

Live Preview

For the home page, I used CSS Grid to place the product types and two banners. For each product page, I used Flexbox since the products were laid out in rows and columns that were of equal dimensions. The item page also uses Flexbox for the product images and the actions buttons below the images.

Flexbox is used for the Navigation bar. The Product link drops down when clicked/hovered over and it uses Flexbox too. When the viewport gets smaller than 769px, a hamburgers menu icon appears which, when clicked, has all the links. The hamburger menu also uses Flexbox.
