Introduction

I wanted to build something using what I had learned about PHP. Something that I would use and that has a real-world application. I use to write all assignments given and their due dates in a notebook. So, I decided to make an application using PHP that would allow me to do all that and also provide more functionality. This project covers the UI design process of the website.

Wireframes

To understand how and where the content and different features would go on the web page, I made wireframes of the three pages. I did this on paper as that is a lot quicker to do. Also, the idea in my head was very vague at this point and pen and paper seemed like the perfect fit to express that idea.
