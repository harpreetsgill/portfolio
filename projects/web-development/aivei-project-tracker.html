Aievei - A project tracking website

Field

Web Development

Tools

HTML

CSS (SASS)

PHP

MySQL

JavaScript

Date

Nov, 2020

Objective

This was a self-led project that I did to understand the CRUD operations in PHP and MySQL. A lot of the code is based on a term project that I did in Term 3.

Listing the possible features

I wrote down the features that I wanted to incorporate into the website. Initially I had planned to make a feature that would allow users to add group projects and share it with other users. But as the deadline got closer, I narrowed the focus to only the essential features.

Structuring the files

To make the development process easier and efficient, I separated the features into different files and directories. All the back-end features were put inside the app directory. Different components of the front-end were split into individual files based on their functionality and put inside the partials directory.

UI Inspiration

I came across this dark UI design. I really liked the colors. I thought of using it in combination with the colors of a code editor to give a feeling of a coding environment.

Wireframing

I made some rough wireframes to see if the design I am imagining can fit the content that is required for this project.

Live Preview

The website allows the user to customize some parts of the code to show how these changes affect the final output.

For the Plugin page, I used fancybox to showcase some of my design work.
