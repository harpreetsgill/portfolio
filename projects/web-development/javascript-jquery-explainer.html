Purpose

This was the final project in Term 2's Web Scripting course.

The purpose was to show an understanding of JavaScript and jQuery. We also were required to incorporate third-party plugins in the website in such a way that they help in explaining the concepts better.

Rough Notes

I made some rough notes of the topics I wanted to explain on the website.

I did this to get an idea of how much space my explanations would take and where I could place them.

UI Inspiration

I came across this dark UI design. I really liked the colors. I thought of using it in combination with the colors of a code editor to give a feeling of a coding environment.

Wireframing

I made some rough wireframes to see if the design I am imagining can fit the content that is required for this project.

Live Preview

The website allows the user to customize some parts of the code to show how these changes affect the final output.
