jQuery

jQuery is a JavaScript library that reduces the amount of code one would otherwise had to write in JavaScript to achieve the same result.

Code

<p>I am a <span id="appClr">red</span> apple.</p>


<script>

$("#appClr").text("");

</script>

Output

I am a red apple.

Selector types

jQuery selectors are how we target the html and css elements.

Some of the ways in which an html element can be targeted are:

  • Identifier: #appClr
  • Classes: .className
  • Tags: div, span
  • Tags can be filtered using their atttributes: input[type="number"]

Code

<p>This paragraph tag has <span id="unique">an identifier</span> and two classes- <span class="odd">first class, </span> <span class="even">second class, </span> <span class="odd">third class</span>.</p>


<script>

$().text("");

</script>

Output

This paragraph tag has an identifier and two classes- first class, second class, third class.

DOM: Document Object Model

Get and Set actions

Value of any element's attribute can be extraced and set using the .attr action.

Code

<p>I am an <span id="appClr" style="color:red;">apple</span>.</p>


<script>

$("#appClr").attr("style", "");

</script>

Output

I am an apple.

Animation Actions

Effects Action

The jQuery library has some built-in effects that can be used to animate the elements.

Three basic effects are:

  1. show and hide
  2. slideUp and slideDown
  3. fadeIn and fadeOut

These pairs can also be toggled using the following actions:

  1. toggle: to show or hide
  2. slideToggle: to slide
  3. fadeToggle: to fade

Code

<p>I can <span id="txtAnmt">ANIMATE</span></p>


<button id="btnSld">Go!</button>


<script>

$("#txtAnmt").("slow");

</script>

Output

I can ANIMATE
